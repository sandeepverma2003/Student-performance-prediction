# Student Performance Prediction System
BCA (Data Science)
5th Semester

## Project Description
This project predicts whether a student will PASS or FAIL based on study hours, attendance percentage, and internal marks using Machine Learning (Logistic Regression).

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Features
- Predicts student result (PASS / FAIL)
- Uses Logistic Regression
- Input validation with defined ranges
- Clean and simple ML implementation
